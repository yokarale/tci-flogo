SDK

TIBCO Cloud Integration - Flogo and TIBCO Flogo Enterprise provide two types of SDK's to build an extension.

  1. Designtime The design-time SDK is used to build the extension user interface code.
  2. Runtime The run-time SDK is used to build the runtime extension code.
